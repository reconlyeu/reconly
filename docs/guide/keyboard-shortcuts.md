# Keyboard Shortcuts

Press `?` at any time (when not typing in a text field) to open the keyboard shortcuts modal.

## Global

| Shortcut | Action |
|----------|--------|
| `?` | Open keyboard shortcuts |
| `Ctrl+K` | Open quick chat |
| `Escape` | Close modal / panel |

## Navigation

| Shortcut | Action |
|----------|--------|
| `G` then `D` | Go to Dashboard |
| `G` then `I` | Go to Digests |
| `G` then `F` | Go to Feeds |
| `G` then `S` | Go to Sources |
| `G` then `T` | Go to Templates |
| `G` then `C` | Go to Chat |
| `G` then `A` | Go to Analytics |
| `G` then `K` | Go to Knowledge Graph |
| `G` then `R` | Go to Feed Runs |
| `G` then `E` | Go to Settings |

## Lists

| Shortcut | Action |
|----------|--------|
| `J` | Next item |
| `K` | Previous item |
| `Enter` | Open selected item |
| `X` | Toggle selection (checkbox) |

## Actions

| Shortcut | Action |
|----------|--------|
| `N` | New / Create |
| `E` | Edit selected |
| `Delete` | Delete selected |
| `/` | Focus search |

> **Note:** Shortcuts are disabled when typing in text inputs, textareas, or contenteditable elements.
