# Reconly Core

Core functionality for Reconly - CLI and content processing.

## Features

- CLI for fetching and summarizing content
- RSS, YouTube, and website fetchers
- Multiple LLM providers (Anthropic Claude, HuggingFace)
- Database models and CRUD operations
- Configuration management

## Installation

```bash
pip install reconly-core
```

## Usage

```bash
reconly fetch https://example.com/rss-feed
```
